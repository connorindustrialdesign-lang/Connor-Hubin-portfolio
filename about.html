About This Site

This website demonstrates a basic three-page layout using only front-end technologies: HTML, CSS, and JavaScript.

The pages share a common header, navigation, and footer. A small JavaScript file provides a few reusable functions, such as highlighting the active page, toggling between light and dark themes, and handling form interactions on the Contact page.

Technologies Used

  • HTML5 for structure.
  • CSS3 for styling and responsive layout.
  • JavaScript for client-side interactivity.
